# Tugas 2 — Styling Website Profil

Tugas praktik untuk mata pelajaran PPLG, kelas XII PPLG SMT 2.

## Deskripsi
Website biodata diri dalam bentuk kartu profil, yang dipercantik menggunakan
styling CSS sesuai instruksi tugas: warna background, font family, ukuran
font, perataan teks, dan warna heading.

## Poin Styling yang Digunakan
- **Background color** — gradasi warna biru gelap pada body halaman
- **Font family** — `Poppins`, dengan fallback `Segoe UI` dan `sans-serif`
- **Font size** — judul (nama) 26px, teks informasi 15px
- **Text align** — judul dan subjudul rata tengah, daftar info rata kiri-kanan
- **Warna heading** — nama ditampilkan dengan warna biru (`#1d4ed8`)

## Cara Menjalankan
1. Buka file `biodata-website.html` menggunakan browser (Chrome, Edge, dll).
2. Tidak perlu koneksi internet karena semua styling ditulis langsung
   di dalam file (inline `<style>`), tanpa dependensi eksternal.
